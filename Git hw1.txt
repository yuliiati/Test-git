1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Команда git clone:  git clone https://github.com/yuliiati/Test-git.git

Предварительно нужно сгенерировать SSh-ключ. 
И настроить конфигурацию: git config --global user.name "User Name"/git config --global -l


2. Как создать git репозиторий из локального каталога на своем компьютере. 
   Как его отправить на github.com?

Выполнить команду - git init. Эта команда создаёт в текущем каталоге новый подкаталог с именем .git, 
содержащий все необходимые файлы репозитория — структуру Git репозитория. 
На этом этапе ваш проект ещё не находится под версионным контролем.

Для того, чтобы отправить на гитхаб, нужно:

- git add .
- git commit -m 'Test project'
- git push


3. Как получить на локальный компьютер изменения с github.com

- git pull 

4. Какую информацию нам дают команда :

- git status - отображает состояние рабочего каталога и раздела проиндексированных файлов. 
- git log - просмотр истории коммитов.
- git remote -v - эта команда используется для просмотра подключенных удаленных репозиториев.
